API密钥

使用此密钥访问音乐API服务

如何使用API密钥

在请求头中添加 X-API-Key 或在URL参数中添加 apiKey:

使用请求头:

fetch('/api/music/url', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    platform: 'netease',
    musicInfo: { id: '1824020871' },
    quality: '320k'
  })
})

使用URL参数:

fetch('/api/music/url?apiKey=YOUR_API_KEY', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    platform: 'netease',
    musicInfo: { id: '1824020871' },
    quality: '320k'
  })
})
